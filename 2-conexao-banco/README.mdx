Conectando ao Banco de Dados
=============================

Este exemplo conecta com um banco de dados local e insere nela uma Produto. 

Adiciona ao exmeplo anterior:

* mongoose - ORM para encapsular a lógica de banco de dados com javascript
* [Docker CE](https://docs.docker.com/install/linux/docker-ce/ubuntu/) - Para trabalhar localmente com máquina virtual

Após instalação do docker e das dependencias rodar (como administrador) a imagem do mongoDB na máquina virtual:

```shell
$ docker run --name mongobd -p 27017:27017 -d mongo                                   
```

Para visualizar o banco de dados criado use [MongoDB Compass](https://www.mongodb.com/products/compass) ou o [Robot 3T](https://robomongo.org/)

Comandos básicos do docker
--------------------------

```shell
#Lista máquinas virtuais em funcionamento:
$ docker ps -a

#Lista todas as máquinas virtuais disponiveis:
$ docker ps -a

#Iniciar máquina virtual
$ docker start [NAME]

#Desliga máquina virtual
$ docker stop [NAME]
```
