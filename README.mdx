NodeJS usando MongoDB
======================

* NodeJS 
* Express
* Nodemon
* [Docker CE](https://docs.docker.com/install/linux/docker-ce/ubuntu/) - Para trabalhar localmente com máquina virtual

Após instalação do docker e das dependencias rodar (como administrador) a imagem do mongoDB na máquina virtual:

```shell
$ docker run --name mongobd -p 27017:27017 -d mongo                                   
```

Para visualizar o banco de dados criado use [MongoDB Compass](https://www.mongodb.com/products/compass) ou o [Robot 3T](https://robomongo.org/)
